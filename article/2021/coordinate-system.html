# GCS (Geographic coordinate system) 地理坐标系

用以表示地球球体上一个具体位置点的空间坐标系统

一个地理坐标系统的描述

Datum 基准面参数

  • 椭球体

根据球心位置 可为地心(质心)或参心

a 长半轴 单位米

b 短半轴 单位米

f 扁率 (扁率是一个较小的值,通常采用扁率的倒数进行存储 扁率计算:f=(a-b)/a 扁率存储:1/f)

第一偏心率 e=(a²-b²)½/a

第二偏心率 e`=(a²-b²)½/b

  • 3参数/7参数

(3平移,1缩放,3旋转)

用于两个椭球体进行转换 默认为向WGS 84椭球体转换的参数

表示方法

大地(经纬度)坐标系:(L,B,H)

空间直角坐标系:(X,Y,Z)

两种坐标可相互转换,在不考虑小数精度的情况下,二者转换不会精度丢失

WGS 84地理坐标系参数(来源QGIS)

WGS 84
WKT
GEOGCRS["WGS 84",
    DATUM["World Geodetic System 1984",
        ELLIPSOID["WGS 84",6378137,298.257223563,   // 椭球体名称  长半轴  扁率的倒数
            LENGTHUNIT["metre",1]]],         // 长度单位  米  每米数值
    PRIMEM["Greenwich",0,                           // 本初子午线  格林威治  0度
        ANGLEUNIT["degree",0.0174532925199433]],    // 角度单位  度  每度数值
    CS[ellipsoidal,2],
        AXIS["geodetic latitude (Lat)",north,     // 纬度 北方向为正
            ORDER[1],
            ANGLEUNIT["degree",0.0174532925199433]],  // 角度单位  度  每度数值
        AXIS["geodetic longitude (Lon)",east,     // 经度 东方向为正
            ORDER[2],
            ANGLEUNIT["degree",0.0174532925199433]],  //  角度单位  度  每度数值
    USAGE[
        SCOPE["unknown"],
        AREA["World"],           //使用区域
        BBOX[-90,-180,90,180]], // 使用经纬度范围
    ID["EPSG",4326]]    // WKID
Proj4
+proj=longlat +datum=WGS84 +no_defs
范围
-180.00, -90.00, 180.00, 90.00
